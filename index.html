Farmhash.Sharp

Farmhash.Sharp

Farmhash.Sharp is an extremly simple, low-level, and blazingly fast library for computing Google's Farmhash algorithm for .NET

The Farmhash.Sharp library can be installed from NuGet: 
PM> Install-Package Farmhash.Sharp

Example

Below are a couple of examples in F# (but Farmhash.Sharp is equally usable from C#, VB.NET, and any .NET language).

#r "Farmhash.Sharp.dll"
open Farmhash.Sharp

printfn "'hello' 32bit hash: %x" (Farmhash.Hash32("hello"))
printfn "'hello' 64bit hash: %x" (Farmhash.Hash64("hello"))

let arr = [| 1uy .. 100uy |]
printfn "byte array 64bit hash: %x" (Farmhash.Hash64(arr, arr.Length))
